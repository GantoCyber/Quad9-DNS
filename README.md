# Quad9-DNS

This script allows you to use Quad9 services in DNS-over-TLS to secure DNS exchanges
### # To install it :
```
### sudo apt update && sudo apt upgrade**
### wget https://raw.githubusercontent.com/GantoCyber/Quad9-DNS/main/Quad9-DNS.sh**
### locate Quad9-DNS.sh
### cd [your path]
### bash Quad9-DNS.sh
```
Congrats & enjoy it :)
